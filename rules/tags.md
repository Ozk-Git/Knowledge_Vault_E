# Tag Rules

Tag system used across the entire Vault.

| Tag | Meaning |
|---|---|
| `#llm-draft` | AI-generated draft, awaiting human review |
| `#wiki` | LLM-Wiki integration page — cross-note synthesis of a concept |
| `#paper-relevant` | Note directly related to an ongoing paper |
| `#open-question` | Unresolved question or item requiring further experiments |
| `#claim` | Idea currently used as, or candidate for, a paper claim |
| `#established` | Knowledge well-supported by literature or experiments |
| `#hypothesis` | A hypothesis you have proposed — unverified |
| `#my-interpretation` | Your own interpretation of data or literature |
